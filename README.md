# Dashboard de Assinaturas XBOX
Conforme desafio proposto, foi criado um dashboard de vendas utilizando a base (fonte de dados) fornecida.
Adicionei às informações iniciais a coluna Year e Month baseadas na coluna original Start Date para facilitar a filtragem dos dados.

Foram feitos 5 gráficos:
1. Most Subscribed Plans: Mostra os planos mais assinados.
2. Auto-Renewing Subscriptions: Mostra quantos planos foram contratados com renovação automática.
3. EA Play Season Pass: Mostra o número de assinaturas do EA Play Season Pass.
4. Minecraft Season Pass: Mostra o número de assinaturas do Minecraft Season Pass.
5. Subscription Plan Prices: Mostra uma comparação de valores entre os planos principais e os planos que podem ser agregados (EA Play e Minecraft).
